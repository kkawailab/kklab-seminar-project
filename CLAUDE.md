# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## リポジトリの概要

これはコードのリポジトリではなく、ゼミ（kklab）の卒業制作作品報告書のためのドキュメントリポジトリである。ビルド・リント・テストのコマンドは存在しない。

中心となるファイルは `graduation_project_template.md` — 卒業制作作品報告書の全20章からなる日本語テンプレート。学生がこれを複製・記入して報告書を作成することを想定している。

## テンプレートの構成と規約

- **言語**: すべて日本語。編集・追記も日本語で行うこと。文体は「〜である／〜してください」調に合わせる。
- **章構成**: 基本情報 → 作品概要 → 背景 → 利用者 → 解決方法 → SFプロトタイピング（第6章）→ 利用シナリオ → 画面 → システム構成 → AI利用（第10章）→ 制作過程 → 評価 → 倫理的・法的・社会的課題（第13章）→ 独自性 → 制約 → 学び → まとめ → 参考文献 → 生成AI利用記録（第19章）→ 付録。
- **削除可能な章**: 第10章（AI・エージェントの利用）はAI未使用の作品では削除可、16.3（チーム活動）は個人制作では削除可。テンプレート内の注記に従う。
- **画像**: `images/` ディレクトリ配下を参照する規約（例: `images/future-scene.png`、`images/main-screen.png`）。ディレクトリ自体はまだ存在しないため、画像を追加する際に作成する。
- **図表**: システム構成図には Mermaid（```mermaid フェンス）を使用。表は GitHub Flavored Markdown のテーブル記法。
- **特徴的な要素**: このテンプレートは通常の技術報告書に加えて「SFプロトタイピング」（未来の物語・バックキャスティング・生成AIによる未来画像とそのプロンプト記録）と「生成AI利用記録」（AIとの対話例・制作者自身の判断の記録）を必須要素として含む。テンプレートを改訂する際もこの構造を維持すること。
